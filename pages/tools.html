tools

To answer the questions I like to ask about the brain and human behavior, I use the tools of data science---a term I think many in my field will become comfortable with as the stigma of "data science" as some sort of trendy competitor fades away and we accept the necessity of computational tools as the method-of-choice for computational questions. I don't mean computational to mean exclusively computational modeling. I mean computational to be that the toy models we use to investigate the brain are growing ever more complex, not so that we can simulate the brain but so that we can control for more and more noise so that we can focus on the dimensions of cognitive behavior.
